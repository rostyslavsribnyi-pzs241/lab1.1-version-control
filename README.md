# Лабораторна робота 1.1
Керування версіями за допомогою Git

## Мета роботи
Вивчити та закріпити на практиці можливості системи керування версіями Git,
отримати практичний досвід використання Git та GitHub.

## Короткий опис
У межах лабораторної роботи створено репозиторій GitHub, 
реалізовано роботу з гілками, додано службові файли,
створено Pull Request та виконано злиття гілок.
