# Лабораторна робота 1.1
Керування версіями за допомогою Git

## Мета
Вивчити та закріпити навички роботи із системою Git, отримати практичний досвід використання GitHub.

## Зміст
- Ініціалізація репозиторію
- Створення гілок
- Робота з віддаленим репозиторієм
- Створення Pull Request
- Додавання Collaborators та Reviewers
